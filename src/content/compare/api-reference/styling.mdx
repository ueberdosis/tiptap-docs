Compare renders its default diff UI with CSS classes. Add your own class or attributes with `displayOptions` when calling `displayChanges`.

```ts
editor.commands.displayChanges({
  changes,
  otherDoc,
  displayOptions: {
    attributes: { class: 'my-removed-content' },
    replacementAttributes: { class: 'my-added-content' },
  },
})
```

## Default classes

```css
/* Content from the displayed document */
.diff-suggestion {
  background: #fee2e2;
}

/* Replacement content from the other document */
.diff-suggestion-diff {
  background: #dcfce7;
}

/* A selected suggestion */
.diff-suggestion--selected,
.diff-suggestion-diff--selected {
  outline: 2px solid #958df1;
}

/* Individual changes inside a grouped change */
.diff-suggestion-sub-change,
.diff-suggestion-diff-sub-change {
  text-decoration: underline;
}
```

Grouped changes also receive `.diff-suggestion--change-group` and `.diff-suggestion-diff--change-group`. Selected sub-changes receive `--selected` variants.

## Custom rendering

Set `displayOptions.renderDecorations` when classes are not enough. The callback receives the suggestion, its range and selection state, plus `defaultRenderDecorations()` so you can adjust the default output or return your own ProseMirror decorations.
