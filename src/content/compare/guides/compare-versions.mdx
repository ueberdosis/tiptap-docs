## Compare the versions

Use the `compareVersions` utility to compare two versions. It takes an older and newer Yjs version update.

When you use the [Snapshot extension](/collaboration/documents/snapshot), version IDs are available in `editor.storage.snapshot.versions`. Each version has a `version` property, which you can pass to `getVersions`. See [Snapshot](/collaboration/documents/snapshot#access-version-data-directly-from-storage) for how to save and fetch versions.

```ts
import { Editor } from '@tiptap/core'
import { compareVersions, getVersions } from '@tiptap-pro/compare'

const editor = new Editor()

const { prevSnapshot: versionA, snapshot: versionB } = await getVersions({
  provider,
  fromVersion: 1,
  toVersion: 2,
})

const changes = compareVersions({
  versionA,
  versionB,
  schema: editor.schema,
})
```

`compareVersions` returns a list of Change objects. Each [Change object](/compare/api-reference/types#changes) contains the positions where the document changed, as well as metadata from the version history when it is available.

## Display the changes

First, create the editor where the diff will be displayed:

```ts
import { Compare, getDocumentFromVersion } from '@tiptap-pro/compare'

const diffEditor = new Editor({
  extensions: [Compare],
})
```

Get a snapshot of the document at versions A and B. Set the editor content to version A, then display the changes.

```ts
const docA = getDocumentFromVersion({
  version: versionA,
  schema: diffEditor.schema,
})
const docB = getDocumentFromVersion({
  version: versionB,
  schema: diffEditor.schema,
})

diffEditor
  .chain()
  .setContent(docA)
  .displayChanges({
    changes,
    otherDoc: docB,
  })
  .run()
```
