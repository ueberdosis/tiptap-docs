## Compare the documents

Use the `compareDocuments` utility to compare two documents. It takes two Tiptap JSON documents.

```ts
import { Editor } from '@tiptap/core'
import { compareDocuments } from '@tiptap-pro/compare'

const editorA = new Editor()
const docA = editorA.toJSON()

const editorB = new Editor()
const docB = editorB.toJSON()

const changes = compareDocuments({
  schema: editorA.schema,
  docA,
  docB,
})
```

`compareDocuments` returns a list of Change objects. Each [Change object](/compare/api-reference/types#changes) contains the positions where the document changed.

## Display the changes

First, instantiate the diff editor, which is the editor where the diff will be displayed:

```ts
import { Compare } from '@tiptap-pro/compare'

const diffEditor = new Editor({
  extensions: [Compare],
})
```

Set the diff editor to document A, then display the changes.

```ts
diffEditor
  .chain()
  .setContent(docA)
  .displayChanges({
    changes,
    otherDoc: docB,
  })
  .run()
```

## Style the changes

Compare adds CSS classes to the original content and the replacement content. Add these styles to display the changes as a red and green diff.

```css
/* Document A: removed or changed content */
.diff-suggestion,
.diff-suggestion > * {
  background-color: oklch(80.8% 0.114 19.571);
  color: oklch(0.396 0.141 25.723);
}

/* Keep formatting correct inside replacement content */
strong .diff-suggestion-diff {
  font-weight: normal;
}

em .diff-suggestion-diff {
  font-style: normal;
}

code:not(pre code) .diff-suggestion-diff {
  font-family: sans-serif;
}

.diff-suggestion-diff strong {
  font-weight: bold;
}

.diff-suggestion-diff em {
  font-style: italic;
}

.diff-suggestion-diff code:not(pre code) {
  font-family: monospace;
}

/* Lighter backgrounds for grouped changes */
.diff-suggestion.diff-suggestion--change-group,
.diff-suggestion.diff-suggestion--change-group > *:not(.diff-suggestion-sub-change) {
  background-color: oklch(0.936 0.032 17.717);
}

.diff-suggestion-sub-change {
  background-color: oklch(80.8% 0.114 19.571);
}

/* Document B: replacement content */
.diff-suggestion-diff,
.diff-suggestion-diff > * {
  background-color: oklch(87.1% 0.15 154.449);
  color: var(--black);
}

.diff-suggestion-diff.diff-suggestion-diff--change-group,
.diff-suggestion-diff.diff-suggestion-diff--change-group > *:not(.diff-suggestion-diff-sub-change) {
  background-color: oklch(0.962 0.044 156.743);
}

.diff-suggestion-diff-sub-change {
  background-color: oklch(87.1% 0.15 154.449);
}

/* Keep table changes inside their table cells */
.diff-suggestion-diff:has(tr) {
  display: contents;
}

.diff-suggestion-diff:has(tr) td,
.diff-suggestion-diff:has(tr) th {
  background-color: oklch(87.1% 0.15 154.449);
}
```

### Colorblind styles

The following stylesheet uses orange and blue to display the diff. The colorblind styles are activated when the diff is wrapped inside an element with the `colorblind-mode` class.

```scss
.colorblind-mode {
  .diff-suggestion,
  .diff-suggestion > * {
    background-color: oklch(83% 0.13 70);
    color: oklch(0.396 0.08 55);
  }

  .diff-suggestion.diff-suggestion--change-group,
  .diff-suggestion.diff-suggestion--change-group > *:not(.diff-suggestion-sub-change) {
    background-color: oklch(0.95 0.05 70);
  }

  .diff-suggestion-sub-change {
    background-color: oklch(83% 0.13 70);
  }

  .diff-suggestion-diff,
  .diff-suggestion-diff > * {
    background-color: oklch(83% 0.1 250);
  }

  .diff-suggestion-diff.diff-suggestion-diff--change-group,
  .diff-suggestion-diff.diff-suggestion-diff--change-group
    > *:not(.diff-suggestion-diff-sub-change) {
    background-color: oklch(0.95 0.035 250);
  }

  .diff-suggestion-diff-sub-change {
    background-color: oklch(83% 0.1 250);
  }

  .diff-suggestion-diff:has(tr) {
    td,
    th {
      background-color: oklch(83% 0.1 250);
    }
  }

  .diff-legend-color--document-a {
    background-color: oklch(83% 0.13 70);
  }

  .diff-legend-color--document-b {
    background-color: oklch(83% 0.1 250);
  }
}
```
