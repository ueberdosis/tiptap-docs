## Install
Quickly add the Simple Editor template to your project using the Tiptap CLI:

```bash
npx tiptap-cli add simple
```

## Usage
After installation, integrate the editor into your React or Next.js application as follows:

```jsx
import { SimpleEditor } from "@/components/templates/simple/components/simple-editor"

export default function Page() {
  return <SimpleEditor />
}
```

## Features

The Simple Editor Template offers quick and straightforward integration, extensive customization through additional Tiptap extensions and UI Components, a fully open-source MIT license ensuring no licensing conflicts, and a responsive design ready for immediate use across various devices.

- Responsive Design: Ready-to-use UI that's responsive out of the box.
- Open Source: Fully MIT-licensed, ensuring no licensing conflicts.
- Basic Formatting: Bold, Italic, Underline
- List Support: Bullet, Ordered, and Task Lists
- Text Alignment: Left, Center, Right, and Justified
- Headings: Multiple levels accessible via dropdown
- Image Upload: Easy-to-use image insertion
- Link Management: Simple interface for adding and editing links
- Undo/Redo: Reliable history management
