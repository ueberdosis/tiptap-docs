Tiptap provides a variety of node components to help you build customizable editors. These packages can be mixed, matched, and installed via our CLI, so you only get exactly what you want.

| Node component                                                          | Description                                            |
|-------------------------------------------------------------------------|--------------------------------------------------------|
| [`Code Block Node	`](/ui-components/node-components/code-block-node)    | Renders code blocks with optional syntax highlighting. |
| [`Image Node`](/ui-components/node-components/image-node)               | Displays images inline or block-level                  |
| [`Image Upload Node`](/ui-components/node-components/image-upload-node) | Allows uploading images directly from your editor.     |
| [`List Node`](/ui-components/node-components/list-node)                 | Provides ordered/unordered list functionality.         |
