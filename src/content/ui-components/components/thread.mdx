Components for displaying and interacting with comment threads.

## Install

```bash
npx tiptap-cli add thread
```

## Usage

```tsx
import { Thread, ThreadFloating } from "@/components/tiptap-ui/thread"
import { useEditor, EditorContent } from "@tiptap/react"
import { Comments } from "@tiptap-pro/extension-comments"

export default function MyEditor() {
  const editor = useEditor({
    extensions: [
      // other extensions
      Comments.configure({
        // comments configuration
      }),
    ],
  })

  return (
    <div>
      <EditorContent editor={editor} />
      <ThreadFloating editor={editor} />
    </div>
  )
}
```

## Props

| Name   | Type           | Default | Description                |
| ------ | -------------- | ------- | -------------------------- |
| editor | Editor \| null | null    | The TipTap editor instance |
