You can see [our guide to custom extensions](/hocuspocus/guides/custom-extensions) to find out how to create your own.

## Table of Contents

We already created some very useful extensions you should check out for sure:

| Extension                                    | Description                                                                    |
|----------------------------------------------|--------------------------------------------------------------------------------|
| [Database](/hocuspocus/server/extensions/database) | A generic database driver that is easily adjustable to work with any database. |
| [Logger](/hocuspocus/server/extensions/logger)     | Add logging to Hocuspocus.                                                     |
| [Redis](/hocuspocus/server/extensions/redis)       | Scale Hocuspocus horizontally with Redis.                                      |
| [SQLite](/hocuspocus/server/extensions/sqlite)     | Persist documents to SQLite.                                                   |
| [Throttle](/hocuspocus/server/extensions/throttle) | Throttle connections by ips.                                                   |
| [Webhook](/hocuspocus/server/extensions/webhook)   | Send document changes via webhook to your API.                                 |
| [S3](/hocuspocus/server/extensions/s3)             | Persist documents to Amazon S3 or any S3-compatible storage service.           |
