The fastest way to get started with the AI Agent extension is with Tiptap Content AI Cloud API. This works out of the box, you only need to provide the authentication credentials to the extension.

<Callout title="Note" variant="info">
  This guide relies on using the [Tiptap Content AI backend
  service](https://cloud.tiptap.dev/v2/cloud/ai). If you want to configure it with your own custom
  LLM, follow the [custom LLMs guide](/content-ai/capabilities/agent/custom-llms) instead.
</Callout>

## Start your Content AI app

1. **Activate a plan:** Begin a [free trial or choose a subscription](https://cloud.tiptap.dev/v2/billing).
2. **Configure your AI app:** Add your OpenAI API key to your [Content AI app](https://cloud.tiptap.dev/v2/cloud/ai).
3. **Set up authentication:** Generate a JWT (HS256 algorithm) with our provided secret to authenticate the extension against our service. [Get your JWT secret.](https://cloud.tiptap.dev/v2/cloud/ai)

## Add your authentication credentials to the extension

Use the AiAgentProvider class to connect with Tiptap Cloud. It handles the communication with Tiptap Cloud and manages the state of the AI Agent.

```tsx
import { Editor } from '@tiptap/core'
import StarterKit from '@tiptap/starter-kit'
import AiAgent, { AiAgentProvider } from '@tiptap-pro/extension-ai-agent'

const provider = new AiAgentProvider({
  // Your Tiptap Content AI app id
  appId: 'APP_ID_HERE',
  // This needs to be your generated JWT and MUST NOT be the OpenAI API key!
  token: 'TOKEN_HERE',
  // Other configuration options...
})
```

When you use the Tiptap Cloud provider to send messages to the AI Agent, the Tiptap Cloud API will be called to get the AI Agent's response. The AI Agent will then generate chat messages and make edits to the document to accomplish the user's task.

## Customize the OpenAI model

You can customize the OpenAI model that is used in Tiptap Cloud by setting the `modelName` option. The AI Agent works best with `gpt-4.1` and `gpt-4o` models, as they provide the best performance for complex document editing tasks. Note that the model needs to support tool calling.

```tsx
const provider = new AiAgentProvider({
  // ...other options
  modelName: 'gpt-4.1',
})
```
