Server-side tools are tools that run on the server. Use them to interact with external data sources. For example:

- Search the web for information
- Query a database
- Call an external API

The guide has different versions depending on the AI library you use.

- [Vercel AI SDK](/content-ai/capabilities/agent/custom-llms/server-side-tools/vercel-ai-sdk)
- [OpenAI Chat Completions API](/content-ai/capabilities/agent/custom-llms/server-side-tools/openai-chat-completions)
