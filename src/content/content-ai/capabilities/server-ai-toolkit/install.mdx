## Basic usage

The Server AI Toolkit requires schema awareness data from a Tiptap editor instance. You can get this data on the client-side or server-side (if you instantiate a Tiptap editor on the server).

### Get schema awareness data

On the client-side, get the schema awareness data from the editor:

```tsx
import { getAiToolkit } from '@tiptap-pro/ai-toolkit'

const editor = useEditor({
  extensions: [StarterKit, AiToolkit],
})

// Get schema awareness data
const schemaAwarenessData = getAiToolkit(editor).getJsonSchemaAwarenessData()
```

Send this data to your server endpoint when making API calls.

### Create the server toolkit

On the server, create a Server AI Toolkit instance with the schema awareness data:

```typescript
import { getServerAiToolkit } from '@tiptap-pro/ai-toolkit/server-ai-toolkit'

const toolkit = getServerAiToolkit({
  schemaAwarenessData,
})

// Read the document
const json = toolkit.getJson()

// Edit the document
toolkit.setJson({
  type: 'doc',
  content: [{ type: 'paragraph', content: [{ type: 'text', text: 'Hello from server!' }] }],
})
```

## With storage

Use Tiptap Cloud storage to load and save documents:

```typescript
import { getServerAiToolkit, TiptapCloudStorage } from '@tiptap-pro/ai-toolkit/server-ai-toolkit'

const toolkit = getServerAiToolkit({
  schemaAwarenessData,
  storage: new TiptapCloudStorage({
    documentIdentifier: 'my-document',
    appId: process.env.TIPTAP_CLOUD_APP_ID,
    apiSecret: process.env.REST_API_SECRET,
  }),
})

// Load document from storage
await toolkit.loadDocument()

// Edit the document
toolkit.setJson({
  type: 'doc',
  content: [{ type: 'paragraph', content: [{ type: 'text', text: 'Updated content' }] }],
})

// Save document to storage
await toolkit.saveDocument()
```

## With AI agent tools

Install tool definitions for your AI framework:

```bash
npm install @tiptap-pro/ai-toolkit-ai-sdk
```

Use the server tool definitions with your AI agent:

```typescript
import { getServerAiToolkit, TiptapCloudStorage } from '@tiptap-pro/ai-toolkit/server-ai-toolkit'
import { serverToolDefinitions } from '@tiptap-pro/ai-toolkit-ai-sdk'
import { streamText } from 'ai'
import { anthropic } from '@ai-sdk/anthropic'

const toolkit = getServerAiToolkit({
  schemaAwarenessData,
  storage: new TiptapCloudStorage({
    documentIdentifier: 'my-document',
    appId: process.env.TIPTAP_CLOUD_APP_ID,
    apiSecret: process.env.REST_API_SECRET,
  }),
  tools: serverToolDefinitions(),
})

const result = streamText({
  model: anthropic('claude-haiku-4-5-20251001'),
  system: `You are an assistant that can edit rich text documents.
${toolkit.getSchemaAwarenessPrompt()}`,
  messages,
  tools: toolkit.getTools(),
})
```
