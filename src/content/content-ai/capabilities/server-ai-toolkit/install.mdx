## Basic usage

The Server AI Toolkit requires schema awareness data from a Tiptap editor instance. You can get this data on the client-side or server-side (if you instantiate a Tiptap editor on the server).

### Get schema awareness data

Create a Tiptap editor instance and get the schema awareness data:

```tsx
import { getSchemaAwarenessData } from '@tiptap-pro/server-ai-toolkit'

// Create a new Tiptap Editor instance
const editor = new Editor()

// Get schema awareness data
const schemaAwarenessData = getSchemaAwarenessData(editor)
```

Then store the schema awareness data in the server, or send it to the server on every API request.

### Create the server toolkit

On the server, create a Server AI Toolkit instance with the schema awareness data:

```typescript
import { getServerAiToolkit } from '@tiptap-pro/server-ai-toolkit'

const toolkit = getServerAiToolkit({
  // The schema awareness data created earlier
  schemaAwarenessData,
})

// Read the document
const json = toolkit.getJson()

// Edit the document
toolkit.setJson({
  type: 'doc',
  content: [{ type: 'paragraph', content: [{ type: 'text', text: 'Hello from server!' }] }],
})
```

## With storage

Use Tiptap Cloud storage to load and save documents:

```typescript
import { getServerAiToolkit, TiptapCloudStorage } from '@tiptap-pro/server-ai-toolkit'

// Create a Server AI Toolkit instance
const toolkit = getServerAiToolkit({
  schemaAwarenessData,
  // Connect to Tiptap Cloud to load and save documents
  storage: new TiptapCloudStorage({
    documentIdentifier: 'my-document',
    appId: process.env.TIPTAP_CLOUD_APP_ID,
    apiSecret: process.env.REST_API_SECRET,
  }),
})

// Load document from storage
await toolkit.loadDocument()

// Edit the document
toolkit.setJson({
  type: 'doc',
  content: [{ type: 'paragraph', content: [{ type: 'text', text: 'Updated content' }] }],
})

// Save document to storage
await toolkit.saveDocument()
```

## With AI agent tools

Install tool definitions for your AI framework (for example, the [AI SDK by Vercel](https://ai-sdk.dev/)):

```bash
npm install @tiptap-pro/ai-toolkit-ai-sdk
```

Use the server tool definitions with your AI agent:

```typescript
import { getServerAiToolkit, TiptapCloudStorage } from '@tiptap-pro/server-ai-toolkit'
import { serverToolDefinitions } from '@tiptap-pro/ai-toolkit-ai-sdk'
import { streamText } from 'ai'
import { openai } from '@ai-sdk/openai'

// Create a Server AI Toolkit instance
const toolkit = getServerAiToolkit({
  schemaAwarenessData,
  tools: serverToolDefinitions(),
})

// Create an AI agent
const result = streamText({
  model: openai('gpt-5.1'),
  system: `You are an assistant that can edit rich text documents.
${toolkit.getSchemaAwarenessPrompt()}`,
  messages,
  // Provide the tools to the AI model
  tools: toolkit.getTools(),
})
```
