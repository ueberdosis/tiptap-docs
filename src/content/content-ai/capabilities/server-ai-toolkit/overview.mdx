The Server AI Toolkit enables AI agents to read and manipulate Tiptap documents on the server without requiring a browser-based editor instance.

Use it to build server-side AI agents that can edit documents through API endpoints, background jobs, or serverless functions. Perfect for AI-powered document processing, automated content generation, and collaborative editing workflows.

## Core Capabilities

The Server AI Toolkit provides server-side equivalents of the AI Toolkit functionality, optimized for server environments where no editor instance is available.

### AI Agent Tools

Pre-built tool definitions that your AI agent can call to interact with documents on the server. These work with popular frameworks like Vercel AI SDK and provide a generic format for other providers.
