Read content in JSON format. Chunked reads help with large documents.

## `getJson`

Return the entire document as Tiptap JSON.

### Returns

`JsonNode`: The entire document as Tiptap JSON

### Example

```typescript
// Read the whole document as JSON
const json = toolkit.getJson()

console.log(json)
// { type: 'doc', content: [...] }
```

## `getJsonNodeRange`

Return JSON content from a specific node range.

### Parameters

- `nodeRangeRequest` (`NodeRangeRequest`): The node range to read
  - `from` (`number`): Start node index
  - `to` (`number`): End node index
- `options?` (`GetJsonNodeRangeOptions`): Options for the `getJsonNodeRange` method
  - `chunkSize?` (`number`): Max characters per chunk. Defaults to 32,000 characters (8000 tokens \* 4 characters per token)

### Returns (`GetJsonNodeRangeResult`)

- `range` (`Range`): The range in the document where the content starts and ends
  - `from` (`number`): Start position in the document
  - `to` (`number`): End position in the document
- `nodeRange` (`NodeRange`): The actual node range that was read
  - `from` (`number`): Start node index
  - `to` (`number`): End node index
- `content` (`JsonFragment`): The JSON content of the nodes

### Example

```typescript
// Read nodes 0 to 10
const result = toolkit.getJsonNodeRange({ from: 0, to: 10 })

console.log(result.nodeRange)
// { from: 0, to: 10 }

console.log(result.content)
// [{ type: 'paragraph', content: [...] }, ...]
```
