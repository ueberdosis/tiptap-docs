Tiptap Shorthand is a token-efficient format for representing Tiptap documents. It reduces AI token costs by up to 80% compared to standard JSON, without sacrificing accuracy.

## How to enable

Add `format: "shorthand"` to your API requests. This parameter is available on all Server AI Toolkit endpoints.

### Get tool definitions

```bash
curl -X POST https://api.tiptap.dev/v3/toolkit/tools \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "format": "shorthand"
  }'
```

### Execute a tool

```bash
curl -X POST https://api.tiptap.dev/v3/toolkit/execute-tool \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "format": "shorthand",
    "tool": "tiptapRead",
    "toolParameters": {
      "startNodeIndex": 0,
      "endNodeIndex": 10
    },
    "document": "..."
  }'
```

### Get schema awareness prompt

```bash
curl -X POST https://api.tiptap.dev/v3/toolkit/schema-awareness-prompt \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "format": "shorthand",
    "schemaAwarenessData": { ... }
  }'
```

## Default behavior

When `format` is not specified, the API defaults to `"json"` for backward compatibility. Existing integrations continue to work without changes.

<Callout title="Format consistency" variant="warning">
  Use the same format value across all API calls in a conversation. Mixing formats within a single AI interaction may cause unexpected behavior.
</Callout>
