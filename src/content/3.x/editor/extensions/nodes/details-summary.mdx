The Details extension enables you to use the `<details>` HTML tag in the editor. This is great to show and hide content.

```bash
npm install @tiptap-pro/extension-details
```

## Usage

```js
import { Details, DetailsSummary, DetailsContent } from '@tiptap-pro/extension-details'

const editor = new Editor({
  extensions: [Details, DetailSummary, DetailsContent],
})
```

## Settings

### HTMLAttributes

Custom HTML attributes that should be added to the rendered HTML tag.

```js
DetailsSummary.configure({
  HTMLAttributes: {
    class: 'my-custom-class',
  },
})
```
