Use Tiptap's `@tiptap-pro/extension-export-markdown` to export your editor's content as a `.md` file. This extension integrates Markdown export functionality into your editor by sending the document to the Tiptap convert service for conversion.

You can also use the [REST API](/conversion/export/markdown/rest-api) instead if you'd prefer to handle the conversion on your end.

## Install the Markdown Export extension

The Conversion extensions are published in Tiptap's private npm registry. Integrate the extensions by following the [private registry guide](/guides/pro-extensions).

Once done you can install and import the **Export Markdown** extension package.

```bash
npm i @tiptap-pro/extension-export-markdown
```

```js
import { ExportMarkdown } from '@tiptap-pro/extension-export-markdown'
```

## Configuring the extension

The `ExportMarkdown` extension can be configured with an `ExportMarkdownOptions` (`object`) as an argument to the `configure` method with the following properties:

```ts
import { ExportMarkdown } from '@tiptap-pro/extension-export-markdown'

const editor = new Editor({
  extensions: [
    // Other extensions ...
    ExportMarkdown.configure({
      endpoint: 'https://api.tiptap.dev/v2/convert',
      token: 'YOUR_TOKEN',
      appId: 'YOUR_APP_ID',
    }),
    // Other extensions ...
  ],
})
```

| Parameter  | Type     | Description                            | Default                               |
| ---------- | -------- | -------------------------------------- | ------------------------------------- |
| `endpoint` | `string` | The Tiptap API endpoint for conversion | `'https://api.tiptap.dev/v2/convert'` |
| `token`    | `string` | The Tiptap JWT for authentication      | `''`                                  |
| `appId`    | `string` | The Tiptap App ID for authentication   | `''`                                  |

## Export a Markdown file

With the extension installed, you can export your editor's content to `.md` using the `exportMarkdown` command.

```ts
/**
 * Export the current document as a Markdown file via the Tiptap convert service.
 *
 * @param options.onExport - Callback function to handle the exported content
 * @example editor.commands.exportMarkdown({ onExport: (context) => {} })
 */
exportMarkdown: (options?: { onExport?: (context: ExportContext) => void }) => ReturnType
```

The `onExport` callback receives an `ExportContext` object:

| Property  | Type     | Description                                              |
| --------- | -------- | -------------------------------------------------------- |
| `content` | `string` | The exported Markdown content as a string (null on failure) |
| `error`   | `Error`  | The error object (null on success)                        |

```js
import { ExportMarkdown } from '@tiptap-pro/extension-export-markdown'

const editor = new Editor({
  extensions: [
    // Other extensions ...
    ExportMarkdown.configure({
      token: 'YOUR_TOKEN',
      appId: 'YOUR_APP_ID',
    }),
    // Other extensions ...
  ],
})

function handleExportMarkdown() {
  editor
    .chain()
    .exportMarkdown({
      onExport(context) {
        if (context.error) {
          console.error('Export failed:', context.error)
          return
        }

        // Download the Markdown file
        const blob = new Blob([context.content], { type: 'text/markdown' })
        const url = URL.createObjectURL(blob)
        const a = document.createElement('a')

        a.href = url
        a.download = 'document.md'
        a.click()

        URL.revokeObjectURL(url)
      },
    })
    .run()
}
```

### How it works

When you call `exportMarkdown`, the extension serializes the editor's document to JSON and sends it to the Tiptap convert service. The service converts the document to Markdown and returns it as a text string. The `onExport` callback receives this string content, which you can then download as a `.md` file or process as needed.
